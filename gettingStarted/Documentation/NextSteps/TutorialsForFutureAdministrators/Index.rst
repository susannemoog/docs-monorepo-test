Tutorials for future administrators
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

`**Modern Template Building**
<http://docs.typo3.org/typo3cms/extensions/doc_tut_templselect/>`_ :
The modern Template Building explains how to use an HTML-Template as
skeleton for your webpage, and helps you with the first steps building
your own website in TYPO3.

`**Futuristic Template Building**
<http://docs.typo3.org/typo3cms/extensions/doc_tut_ftb1/>`_ : TYPO3 uses
the concept of columns to define where the content is rendered. If
your website design does not fit into columns or you need to be very
flexible with the content elements, the extension TemplaVoila may help
you build very flexible designs with TYPO3. This concept is explained
in this tutorial.

`**TypoScript in 45 Minutes**
<http://docs.typo3.org/typo3cms/TyposcriptIn45MinutesTutorial/>`_ :
TypoScript is the powerful configuration language, but it is not a
scripting language. This tutorial explains the basic concepts of
TypoScript and gives a first overview about the most important
functions and objects.
