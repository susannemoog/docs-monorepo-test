Next steps
----------

TYPO3 has a lot of documentation available in the documentation
library. They are divided into four sections. The tutorials section
coversdocuments which provide you with a step-by-step introduction to
a topic. If you need documentation about installing TYPO3 or related
software you will find it in the section Installation. The Core
Documentation isrelated to the core of TYPO3 and mainly describe
behind the scenes technology. Target group of this documentation are
extension- and core developers. In the section Extension Manuals there
is a list of all manuals which are available from extension. That are
a lot of different documents, so for beginners it is sometimes
difficult to find the right document to get started. So, we recommend
the following documents for further reading.
