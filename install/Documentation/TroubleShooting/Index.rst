Trouble Shooting
----------------

The following sections contain some hints, which should help you solve
common problems, which you might encounter. Note that you should also
check the section "System Environment" in the Install Tool. In this
section TYPO3 will inform you about errors and warnings, which were
found in your installation and which might influence performance.
Follow the advices given there to fix those issues! This helps to
solve or even prevent most trouble.
